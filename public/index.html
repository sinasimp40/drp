What does this do?

Instead of Roblox being locked to one PC's installation, you place all your Roblox files into a single folder. The launcher reads from that folder and runs Roblox directly — making it fully portable. Take it on a USB drive, a network share, or any folder you choose.

How to set it up

1

Download the scripts

Download the launcher package below. It includes the setup script and the launcher script.

2

Run Setup

Run SetupPortableFolder.bat — it creates the RobloxPortable\ folder structure automatically.

3

Copy Roblox files

Go to %LOCALAPPDATA%\Roblox\Versions\[hash]\ and copy all files into the RobloxFiles\ folder.

4

Launch!

Run Launch Roblox.bat from your portable folder. Roblox starts from your folder — not the system install.

Folder Structure

📁 RobloxPortable\
🚀 Launch Roblox.bat ← Run this!
⚙️ SetupPortableFolder.bat
📁 RobloxFiles\ ← Your Roblox files go here
▪ RobloxPlayerBeta.exe
▪ *.dll files
▪ (all other Roblox files)
📁 Logs\ ← Launch logs
📁 Cache\ ← Roblox data (stays portable)

Where to find your Roblox files

Open File Explorer and paste this path into the address bar:

%LOCALAPPDATA%\Roblox\Versions

Inside you'll see one or more folders with a hash name (like version-abc123def456). Open the latest one and copy all files into your RobloxFiles\ folder.

Good to know

💾
Data stays portable

The launcher redirects Roblox's AppData to the Cache\ folder inside your portable directory, so your settings travel with the folder.

🔄
Roblox updates

When Roblox updates, you'll need to copy the new version files into your RobloxFiles\ folder to keep up.

🖥️
Windows only

The launcher scripts are Windows batch files (.bat). They run on any Windows 10 or Windows 11 machine without installing anything extra.

📋
Launch logs

Every time you launch, a log file is saved in Logs\ with the time and details — useful for troubleshooting.
